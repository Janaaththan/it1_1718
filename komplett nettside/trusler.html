I den digitale verden møter vi mange trusler. Det er ikke alle som er like ærlige og blant annet finner vi tyver, svindlere og terrorister også i den digitale verden. Teknologien er heller ikke alltid til å stole på. For eksempel kan datamaskiner kollapse og data kan gå tapt. Hvert år skjer det mange datainnbrudd og virusangrep, og antallet øker hvert år. Konsekvensene av slike innbrudd kan være store for mange virksomheter, og i verste fall kan det medføre konkurs. Data og hemmeligheter kan også forsvinne til konkurrenter eller andre ondsinnede personer. Når vi skal utrede tiltak mot disse truslene, bør vi derfor kartlegge hvilke trusler som virksomheten kan regne med og spesielt viktig hvilke trusler som er mest alvorlige for virksomheten.

Mange vet ikke hva som er ulovlig og de vet ikke til de store skadene datakriminaliteten kan føre til. Datasnoking og sabotasje eller hacking er straffbart. Datasnokere er personer som kan mye om data og som prøver å trenge seg inn i datasystemer som de kanskje ikke har tilgang til. De kan da endre på data slik at de for eksempel kan endre karakterer, overføre penger til egen konto osv. Dette kalles da for datafiksing. Datalekkasjer skjer når snokerne kopierer eller fjerner data fra en virksomhet som for eksempel kan selges til konkurrenter. Snokerne kan også gjøre endringer i programmer som for eksempel kan slette data på harddisker eller gjøre annen skade.
